# SMB IT Ops Onboarding

## Using /ask
Use the `/ask` endpoint for runbook guidance and policy summaries.

## Provide These Details
- System name and environment (prod/staging/dev).
- Timeframe and alert or ticket ID.
- Error message or symptom.
- Relevant document or section name if known.

## Example Request
"Using /ask, how do I verify last night's backup for the billing-db?"
