# SMB IT Ops FAQ

## How do I request a backup verification?
Include the system name, environment, and the backup job ID or timestamp.

## What if the restore drill fails?
Stop and escalate. Provide logs and the failing step in the checklist.

## Can the assistant change production settings?
No. It can only summarize runbooks and suggest safe next steps.

## How should I update tickets?
Summarize impact, actions, and next steps. Include evidence links.
